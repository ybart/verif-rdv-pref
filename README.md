# verif-rdv-pref

Check prefecture availability for appointments.

## Installation

* Install Crystal
	```
	brew update
	brew install crystal
	```
* Clone the repo locally
	```
	git clone git@github.com:ybart/verif-rdv-pref.git
	cd verif-rdv-pref
	```
* See usage

## Usage

```
shards install
crystal build --release src/web.cr
./web
```
